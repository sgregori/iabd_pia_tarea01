# Creación de un repositorio en Github:

Este repositorio, corresponde a una tarea del C.E. I.A.B.D.

El objetivo es aprendamos a crear un repositorio desde cero e incluir:
- Un archivo `README.md` escrito en lenguaje de marcación. (éste mismo fichero)
- Un archivo en Python llamado `hola_mundo.py` que muestre por pantalla el texto "Hola Mundo".

Este ejercicio es una primera toma de contacto con los repositorios Git y GitHub concretamente, aprendiendo a documentar un proyecto y a subir archivos, etc.
